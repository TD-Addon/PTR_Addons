# PTR Addons

This repository contains MGE and Lua addons for individual PTR projects/provinces.
These addons should be shipped as part of the main content of a release.

## Tamriel Rebuilt

## Distant land
Includes an MGE overrides file for distant land as well as an OpenMW plugin that does the same thing.
These files ensure constructable locations in TR show up properly in distant land.

Currently covers
- Assault on Lake Coronati
- Firemoth + Sulfurwatch
- Khalaan
- Nav Andaram
- Omaynis, The Kwama's Scuttle
- Stendarr Tower Haunting

## Incompatible mods
Includes Lua scripts for MWSE and OpenMW to warn people if they are using other (incompatible) Firemoth mods without using the remover plugin.
